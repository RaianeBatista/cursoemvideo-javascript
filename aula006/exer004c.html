Operadores

Ordem de Precedência dos Operadores de Aritmética

  1. Primeiro resolvemos os parênteses: ()
  2. Segundo resolvemos as potências: **
  3. Mesma ordem de precedência, resolvemos da esquerda para direita, quem aparecer primeiro
      Resolvemos:
      multiplicação *;
      divisão /;
      resto da divisão %
  4. Soma + e subtração -

Operador de Atribuição

Atribuição Simples

var a = 5 + 3 = 8

var b = a % 5 = 3

var c = 5 * b ** 2 = 45

var d = 10 - a / 2 = 6

var e = 6 * 2 / d = 2

var f = b % e + 4 / e = 3

Auto atribuição

O Operador +=, é usado quando a mesma variavel recebe novo valor mais valor dela mesma, sendo acumulativo

var n = 3 =>

n = n + 4 => n+= 4

n = n - 5 => n-= 5

n = n * 4 => n*= 6

n = n / 2 => n/= 2

n = n ** 2 => n+= 2

n = n % 5 => n+= 5

Operadores Relacionais

  • > maior que
  • < menor que
  • >= maior ou igual
  • <= menor ou igual
  • == igual
  • != diferente

Exemplo:

5 > 2 => true

7 < 4 => false

8 >= 8 => false

9 <= 7 => false

5 == 5 => true

4 != 4 => false

Operador de Identidade/De igualdade restrita

Nesse tipo de operador usa-se 3 sinas de iguais, observe o exemplo

5 == 5 => true

5 == '5' => true (Observe que o Javascript, considera essa condição verdadeira, porém, não é verdade)

5 === '5' => false . Aqui o javascript, verifica se as variaveis são do mesmo valor e do mesmo tipo

typeof 5 = number

typeof '5' = string

5 === 5 => true. Valores idênticos

Operadores Lógicos

  • ! negação
  • && conjunção
  • || disjunção

Negação
  • ! true => false (Não verdadeiro = falso)
  • ! false => true (Não falso = verdadeiro)

Conjunção - Operador Binário. Aqui os dois precisam ser verdadeiro para o resultado ser verdadeiro

  • true && true = true
  • true && false = false
  • false && true = false
  • false && false = false

Disjunção - Operador Binário. Aqui basta que um seja verdadeiro para os dois serem verdadeiro.

  • true || true = true
  • true || false = true
  • false || true = true
  • false || false = false

Exemplo de operadores aritméticos, relacionais e lógicos na mesma expressão:

    Ordem de execução:
  1. Operadores Aritméticos: () ** /
  2. Operadores Relacionais(Entre si não ha ordem de precedencia, quem aparecer primeiro da esquerda para a direita): < > >= <=
  3. Operadores Lógicos. Ordem de precedência: 1º ! não ; 2º && e ; 3º || ou
var a = 5
var b = 8
a > b && b % 2 == 0 => Resultado = false

a <= b || b / 2 == 2 => Resultado = true

Caso em alguma expressão tenha a seguinte ordem: &&(e), ||(ou), !=(nao). Teremos a seguinte ordem de execução:

  1. != (Não)
  2. && (e)
  3. || (ou)

Operador Ternário ? :

Se chama Operador ternário, por possuir 3 partes:

  • bloco de teste lógico (true or false)
  • bloco verdaeiro
  • bloco com resultado

teste ? true : false

Exemplo:

var media = 5.5

Se a média for maior que 7, Aprovado. Senão, Reprovado.

media > 7 ? "APROVADO" : "REPROVADO"

Resultado: Reprovado

media += 3

media > 7 ? "APROVADO" : "REPROVADO"

Resultado: Aprovado

Outro exemplo:

var x = 8

var res = x % 2 == 0 ? 5 : 9. Resultado = 5

Outro exemplo:

var idade = 19

var r = idade >= 18 ? "Maior" : "Menor". Resultado = Maior

idade-=3 => 16
